This project is a complete login and registration system in C# ASP.NET. **MVC** architecture and **Entity Framework** is used for this project. 
The main porpose of this project is for learning purposes. 

This project includes,

 1. Registration Page with **email validation** & **password Hashing**
 2. Email Verification Page for **account verification**
 3. Login page with **remember me option** and Logout page
